# 路由结构 #
```
.
├── home                   // 首页
├── login                  // 登录页面
├── register               // 注册页面
├── notFound               // 访问的页面不存在页面
├── noPermission           // 访问的页面没有权限页面
├── blog                   // 用户的博客页面
│   ├── articleCreate      // 创建博文页面
│   ├── articleEdit        // 博文修改页面
│   └── articleView        // 博文详情页面
├── blogConfig             // 博客管理页面
├── accountInfo            // 用户的主页页面
│   ├── accountNews        // 动态消息页面
│   └── accountConfig      // 账户设置页面
└── system                 // 系统管理页面


```
